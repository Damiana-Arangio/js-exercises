<h1 align="center">JS Exercises</h1>

Raccolta di piccoli esercizi in JavaScript che lavorano su numeri, stringhe e array.

## Cartelle incluse

- **js-somma-numeri**  
  Calcolo della somma di una serie di numeri e stampa del risultato.

- **js-max-di-due-numeri**  
  Determina il valore più grande tra due numeri e mostra il risultato.

- **js-parole-length**  
  Confronto della lunghezza di due parole con stampa di quella più lunga
  (oppure entrambe se hanno la stessa lunghezza).

- **js-array-dispari**  
  Gestione di un array numerico con creazione di un nuovo array contenente solo i numeri dispari.

## Concetti principali

- Utilizzo di condizioni (`if`, `else`) per confrontare valori.
- Uso dei cicli per scorrere insiemi di dati.
- Manipolazione di array (creazione, lettura e filtraggio degli elementi).

## Tecnologie utilizzate

- JavaScript
